# electronicshopmanagment
Overview

The Electronic Shop Management System is a Windows-based application built using C#.NET and SQL Server. This project helps manage electronic product inventory, sales, customer details, and billing efficiently.

Features

User Authentication: Admin and staff login system.

Product Management: Add, update, delete, and view products.

Inventory Tracking: Monitor stock levels and generate alerts.

Customer Management: Store and manage customer details.

Sales & Billing: Generate invoices and manage transactions.

Reports & Analytics: Sales reports and stock reports.

Technologies Used

Frontend: C#.NET (Windows Forms)

Backend: SQL Server

IDE: Visual Studio

Database Connectivity: ADO.NET
